# Reliability & Predictive Maintenance Analysis

## Overview
Python-based analysis of the NASA Bearing Run-to-Failure dataset, 
demonstrating automated fault detection techniques applied to 
real industrial sensor data.

## Author
Nathan Kinert | 25+ years Industrial Reliability | MLA | MLT  
IIoT Sensor Implementation | O&G | Offshore Drilling

## What This Project Does
- Loads and processes 984 time-series vibration snapshots
- Calculates RMS and Kurtosis trends across 4 bearings simultaneously
- Implements relative alert thresholds based on individual bearing baselines
- Performs FFT frequency analysis comparing healthy vs degraded signatures
- Generates interactive dashboards with real timestamps

## Tools & Libraries
Python | Pandas | NumPy | SciPy | Plotly

## Dataset
NASA Bearing Run-to-Failure Dataset via Kaggle
